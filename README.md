# Weather Dashboard

## Purpose
 A minimalist weather dashboard that allows you to search the 5-Day forecast for a city of your choice. 

## Built With 
* JavaScript
* HTML
* CSS
## Including
* Bootstrap
* Moment.js
* Google Fonts
* Open Weather API
